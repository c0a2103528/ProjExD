# 第5回
## あぶない！こうかとん（ex05/dagerous_kokaton.py）
### ゲーム概要
- ex06/dangerous_kokaton.pyを実行すると，600x900のスクリーンに溶岩と地面が描画され，こうかとんを移動させ飛んでくる爆弾を避けるゲーム
- こうかとんが爆弾と接触するとゲームオーバーで終了する
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- 爆弾:爆弾が四方向から飛んでくる。
- 再始動:画面端に行った爆弾は消え、新しく爆弾が飛んでくる
